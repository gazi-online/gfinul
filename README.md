# Gazi online

Digital gateway project built with React, TypeScript, Vite, and Supabase.

## Setup

1. Install dependencies:

```bash
npm install
```

2. Create a local env file from `.env.example` and add your Supabase credentials.

3. Start the dev server:

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Database

SQL helpers and schema files are included in the repo root:

- `schema_v1.sql`
- `update_admin_management.sql`
- `update_admin_policies.sql`
- `update_checkout_rls.sql`
- `update_service_requests.sql`
